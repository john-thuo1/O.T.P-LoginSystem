Steps to take

Step 1 : Sign Up

  • Click on Sign up to create account
  • Enter your First Name
  • Enter your Last Name
  • Enter your Username
  • Enter your Phone Number i.e Start with your country code e.g. +254743106974
  • Click on "Submit" button to sign up
  • Your default password is 123456
Step 2 : Test OTP login
  • Click on the "Log In" menu
  • Enter your Username
  • Enter your default Password
  • An SMS message with a unique code will be sent to the simulator
  • Open the message in the simulator and copy the code
  • Enter the code in the O.T.P. page and submit
  • If th code is correct you will be logged in

Africa's Talking Simulator


Initialise Simulator:

  • Click here to open Africa's Talking simulator
  • Enter the mobile number you used to sign up in the simulator
  • Open "Messages" App on the simulator to view incoming messages
